## 🏆 Featured Projects

### 1️⃣ MotionSense VLM  
Video action recognition combining CLIP-based appearance features with Farneback optical flow for motion.  
**Tech:** Python, PyTorch, OpenCLIP, OpenCV  
**Highlights:** Validation accuracy improved from 76.5% → 86.6% on UCF101 10-class subset. Includes modular YAML configs, training scripts, evaluation, and demo.  

### 2️⃣ AIssist – Real-Time AI Meeting Agents  
Full-stack SaaS platform where AI agents join video calls, transcribe sessions, and generate GPT-4o summaries.  
**Tech:** Next.js, React, TypeScript, tRPC, Neon Postgres, OpenAI Realtime API  
**Highlights:** Event-driven architecture, background job processing, OAuth authentication, Vercel deployment, modular agent management.

### 3️⃣ Elastic Face Recognition  
Distributed, autoscaling cloud pipeline for real-time face recognition.  
**Tech:** Python, Flask, AWS EC2, S3, SQS, PyTorch  
**Highlights:** Up to 20 EC2 instances handling 50+ concurrent requests, stateless inference workers, S3-backed queue-based communication.

### 4️⃣ Serverless Video-to-Face Recognition  
Event-driven AWS Lambda pipeline for extracting frames, detecting faces, and recognizing identities.  
**Tech:** Python, Docker, AWS Lambda, S3, PyTorch, OpenCV, MTCNN  
**Highlights:** Reduced compute costs by 35%, supports 1,000+ concurrent video streams/day, automated serverless scaling.

### 5️⃣ Machine Unlearning in Small Language Models  
Selective knowledge removal in LLMs while preserving generalization.  
**Tech:** PyTorch, HuggingFace Transformers, PEFT (LoRA), DPO  
**Highlights:** Llama-3.2-3B achieved strong forgetting (ROUGE-L 0.15), full reproducible pipeline with evaluation on TruthfulQA.

### 6️⃣ ASL Recognition System  
YOLOv5 + I3D pipeline for American Sign Language recognition from video.  
**Tech:** Python, PyTorch, OpenCV  
**Highlights:** mAP@0.5 = 0.87, Top-5 accuracy = 79%, optimized for low-latency inference on WLASL dataset (up to 2000 classes).

### 7️⃣ Audible YouTube Content Summary Generator

**Tech:** Python, HuggingFace Transformers, spaCy, gTTS, youtube_transcript_api  
**Highlights:** High-fidelity summarization with ROUGE-1 = 0.993, supports TTS playback.

### 8️⃣ Sentiment-Based Stock Market Analysis
