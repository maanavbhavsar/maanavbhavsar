## 🏆 Featured AI Systems

### 🎥 MotionSense VLM

Multimodal video action recognition combining CLIP-based appearance features with optical flow.

- **Tech:** PyTorch • OpenCLIP • OpenCV
- **Impact:** Validation accuracy improved from 76.5% → 86.6% on UCF101 10-class subset
- **Highlights:** Modular YAML configs, reproducible training pipeline, demo-ready inference

### 🤖 AIssist – Real-Time AI Meeting Agents

AI agents that join video calls, transcribe sessions, and generate GPT-powered summaries.

- **Tech:** Next.js • TypeScript • tRPC • Neon Postgres • OpenAI Realtime API
- **Highlights:** Event-driven architecture, background job processing, OAuth authentication, modular agent orchestration

### ☁️ Elastic Face Recognition (Distributed ML)

Autoscaling inference pipeline for real-time face recognition.

- **Tech:** Python • Flask • AWS EC2 • S3 • SQS • PyTorch
- **Impact:** Up to 20 EC2 instances, 50+ concurrent requests, stateless worker design, queue-backed distributed processing

### 🧠 Machine Unlearning in Small LLMs

Selective knowledge removal while preserving generalization.

- **Tech:** PyTorch • HuggingFace • PEFT (LoRA) • DPO
- **Impact:** Llama-3.2-3B achieved strong forgetting (ROUGE-L 0.15), fully reproducible evaluation, benchmarked on TruthfulQA

### ⚡ Serverless Video-to-Face Recognition

Event-driven AWS Lambda pipeline for scalable video processing.

- **Tech:** Python • Docker • AWS Lambda • S3 • MTCNN • PyTorch
- **Impact:** Reduced compute costs by 35%, supports 1,000+ video streams/day, fully automated scaling
